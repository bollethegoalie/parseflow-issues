# Parseflow Issues

This repository is the public place to report issues and request improvements for **Parseflow**.

## What to use this for
- Bug reports
- Feature requests / suggestions
- Documentation fixes or example requests
- General questions (non-sensitive)

## What NOT to post here
Please **do not** include:
- API keys (RapidAPI keys, `X-BYOK-Key`, or any secrets)
- Private documents or customer data
- Billing details that include personal information

## Preferred support channel
For anything private (billing/account issues, sensitive documents, security concerns), email:

**hello@parseflow.tech**

## Helpful info to include in an issue
- Endpoint (example: `POST /v2/process`)
- Mode (`deterministic` or `byok_assisted`)
- Request headers used (redact secrets)
- Minimal reproducible input (redact sensitive data)
- Full error response + status code
- `request_id` (if returned)
